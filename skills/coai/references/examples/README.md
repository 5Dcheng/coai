# Examples

## Purpose

This directory contains illustrative examples for CoAI assets.

Use examples to understand shape, density, and style.

## Boundary

- use examples for understanding, not as canonical generation sources
- read examples only after templates if you still need a concrete sample
- do not copy examples verbatim into a real repository without adapting them

## Files

- `feature-example.md`: example feature document
- `mapper-example.json`: example mapper file
- `node-example.json`: example node file
